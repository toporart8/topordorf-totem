# Инструкция по добавлению изображений

Положите ваши фотографии товаров в эту папку (`public/images`).

Имена файлов должны соответствовать тем, что прописаны в `src/utils/products.js`:

1.  `axe_wolf.jpg` - для Топора 'Волчий'
2.  `axe_bear.jpg` - для Топора 'Медвежий'
3.  `figure_falcon.jpg` - для Фигурки 'Сокол'
4.  `idol_makosh.jpg` - для Идола 'Макошь'
5.  `amulet_general.jpg` - для Амулета
6.  `axe_raven.jpg` - для Топора 'Мудрость Ворона'

**Формат:** JPG или PNG.
**Размер:** Желательно, чтобы картинки были примерно одного размера (например, 800x600 или квадратные 600x600), чтобы они ровно смотрелись в карточке.
